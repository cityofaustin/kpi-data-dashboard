Welcome!

This temporary website is part of a City of Austin open data project. It's for helping staff gather information about the City's data resources. Complete documentation about the entire project can be found here.

Who is this site for?
  • City staff who have been tasked with completing surverys.
  • Project staff who are supporting survey takers.
  • Anyone who loves Austin and/or has interest in watching this project unfold.
What's this project all about?

We want to discover more open data resources... so we got a list of 211 Department Key Performance Indicators (KPIs) from Financial Services and created surveys with them. We hope for two main outcomes; that the Open Data Initiative would uncover new open data opportunities to pursue, and the Performance Office would gain new information about how departments prepare specific performance measures. The data we collect will be shared and made available for public download by project end.

Where will the survey results be published?

Probably in a number of places which have yet to be determined. In the meantime, all project code and working datasets can be found on at this project's github repo. We will deliver more direct links to the final results dataset by project end, which will be November 14, 2016.

What will happen to this site when the project is over?

So far, we don't have plans to host this site after the project is completed. However, the source code used to create this site, along with the data we collected, will continue to be available for public review and download on github.
